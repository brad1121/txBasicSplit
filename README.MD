# TX Basic Splitter

Set config.json, including bitcoin rpc details, an output address and number of utxos to create.

```
{
  "host":"127.0.0.1",
  "port":9332,
  "rpcuser": "USER",
  "rpcpassword" : "password",
  "rpctimeout" : 15000,
  "utxostogenerate" : 1000,
  "toaddress" : "bitcoinsvaddress"
}
```

## Install

```npm install```

## run

```npm start```
